# LLM Lab

A collection of hands-on examples and experiments for learning Large Language Models (LLMs).

## Topics Covered

- Prompt Engineering
- Tokens & Tokenization
- Context Windows
- Temperature & Sampling
- System/User Prompts
- Text Summarization
- Structured Outputs
- Function Calling
- Embeddings
- Vector Databases
- RAG (Retrieval-Augmented Generation)
- Chat Memory
- OpenAI API
- Claude API
- Gemini API
- OpenAI-Compatible APIs
- Streaming Responses
- Cost Tracking
